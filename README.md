# Decode Labs - Project 1: Go Build (Frontend Development)

A fully responsive and visually appealing user interface built from scratch. This project demonstrates strong frontend fundamentals, focusing on semantic architecture, modern styling, and interactive elements.

## Features
- **Responsive Design:** Optimized for various screen sizes and devices.
- **Semantic Architecture:** Clean and accessible HTML structure.
- **Custom Styling:** Implemented a specific, cohesive color palette and modern UI patterns.
- **Interactive UI:** Vanilla JavaScript utilized for functional user interactions.

## Tech Stack
- **Markup:** HTML5
- **Styling:** CSS3
- **Logic & Interactivity:** Vanilla JavaScript
